# Happy Burgy App

Fullstack project for a burger restaurant.

## Tech Stack

- React 19  
- TypeScript  
- Tailwind CSS 4  
- React Router 7  
- Node.js  
- Express 5  

## Front-end Setup

1. Go to the front-end directory: `cd front-end`  
2. Install dependencies: `npm install`  
3. Start development server: `npm run dev`  

## Back-end Setup

1. Go to the back-end directory: `cd back-end`  
2. Install dependencies: `npm install`  
3. Start development server: `npm run dev`
